# Trie Data Structure
This is my submission for the slingshot take home assessment, create a Trie data structure.

## Installation
1. Clone the repo:
   `git clone https://github.com/eRuaro/slingshot-trie`
2. Install modules:
    `pip install fastapi, typer, pytest`
